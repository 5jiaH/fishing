# 珠江水系 / 东江 — 上下游核对

## 水系地位

- **所属水系**：珠江流域，珠江三大水系之一（西江、北江、东江）
- **发源地**：江西省寻乌县桠髻钵山（寻乌水）/ 安远县三百山（安远水），两源于龙川合流后称东江
- **入海/汇入点**：经惠州、东莞，于东莞石龙分东江北干流和东江南支流，汇入狮子洋
- **干流长度**：562 km（含上源寻乌水）
- **集雨面积**：35,340 km²
- **流经区域**：江西寻乌/安远 → 广东龙川 → 河源 → 惠州 → 东莞 → 狮子洋入海
- **经纬度范围**：约 113.4°–115.8°E, 22.5°–25.0°N
