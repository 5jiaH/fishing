# 北江 — 上下游核对

## 水系地位

- **所属水系**：珠江流域第二大水系
- **发源地**：江西省赣州市信丰县小茅山山凹出水点（114.614°E, 25.367°N）
- **汇入点**：佛山市三水区思贤滘 → 珠江三角洲网河区 → 虎门出海
- **上源**：浈江（信丰→南雄→始兴→韶关），至韶关沙洲尾汇武江后始称"北江"
- **干流长**：468 km（至思贤滘），573 km（至虎门出海）
- **集雨面积**：46,710 km²（至思贤滘），其中广东省内 42,930 km²
- **年均径流量**：510 亿 m³
- **同名河流**：无混淆（"北江"唯一指珠江水系此河）

### 干流分段

| 段 | 范围 | 说明 |
|---|---|---|
| 上游 | 信丰源头 → 韶关沙洲尾 | 称浈江，汇武江后称北江 |
| 中游 | 韶关沙洲尾 → 清远飞来峡 | 两岸低山丘陵 |
| 下游 | 飞来峡 → 三水思贤滘 | 平坦开阔，堤防保护 |
| 三角洲 | 思贤滘以下 | 西北江三角洲网河区 |

## 数据重建记录

### 原始问题

1. **九曲河.js 混合错误**：原含 64 段，其中 49 段坐标在 115°E 附近（属韩江水系兴宁/五华九曲河），仅 15 段属于北江流域（清远九曲河，112.89°-113.02°E）。已清理。
2. **甘竹河.js 混合错误**：原含 19 段，其中 2 段坐标在 114.71°E（属东江惠州甘竹河），17 段属于北江三角洲。已清理。
3. **缺失一级支流**：墨江、锦江、南花溪、烟岭河、凤岗河、滨江河上游数据均缺失，从天地图 HYDL 补充。
4. **命名不一致**：翁江 = 滃江（异写），浜江 = 滨江（异写），小北江 = 连江（别名）。
