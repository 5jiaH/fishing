# 南渡河 — 上下游核对

## 水系地位

- **所属水系**：南海水系，**独流入海**
- **又名**：擎雷水
- **地位**：雷州半岛唯一一条集水面积 > 1000 km² 的河流
- **发源地**：遂溪县坡仔
- **入海点**：雷州市双溪口入南海雷州湾
- **干流长度**：88 km
- **集雨面积**：1444 km²
- **经纬度范围**：约 109.7°–110.3°E, 20.5°–21.2°N
