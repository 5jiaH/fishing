# 榕江 — 上下游核对

## 水系地位

- **所属水系**：独流入海（粤东沿海第二大河流）
- **发源地**：南河发源于汕尾陆河凤凰山，北河（汤溪）发源于丰顺北斗镇
- **入海点**：南河与北河于揭阳榕城炮台双溪嘴汇合后入汕头牛田洋注南海
- **干流长度**：175–196 km（南河/干流），北河 92 km
- **经纬度范围**：约 115.8°–116.8°E, 23.2°–23.8°N
