# 流溪河 — 上下游核对

## 水系地位

- **所属水系**：珠江三角洲河网 — 北江流域区域河流（非北江干流一级支流，也非独立水系）
- **发源地**：广州从化区吕田镇与韶关新丰县交界处
- **归宿**：白云区江村 → 白坭河 → 珠江三角洲 → 南海
- **干流长度**：156–157 km
- **流域面积**：2290–2300 km²
- **经纬度范围**：约 113.2°–113.8°E, 23.3°–23.9°N

> 流溪河属北江流域管辖，但非北江支流，也非独立入海水系。本文件夹暂按既有位置保留。
