# 深圳河 — 上下游核对

## 水系地位

- **所属水系**：珠江三角洲水系 / 珠江口水系，独流入深圳湾
- **发源地**：梧桐山、牛尾岭
- **入海点**：独立汇入深圳湾（后进入伶仃洋）
- **干流长度**：37 km
- **流域面积**：312.5 km²（深圳侧 187.5、香港侧 125）
- **特点**：下游为深港界河
- **经纬度范围**：约 114.0°–114.25°E, 22.45°–22.65°N
