# 潭江 — 上下游核对

## 水系地位

- **所属水系**：独流入海（珠三角水系一级河流），经银洲湖在崖门口汇入黄茅海入海
- **发源地**：恩平市天露山
- **入海点**：银洲湖 → 崖门口 → 黄茅海
- **干流长度**：248 km
- **集雨面积**：5068 km²
- **经纬度范围**：约 112.0°–113.2°E, 22.0°–22.8°N
