# 练江 — 上下游核对

## 水系地位

- **所属水系**：独流入海（粤东沿海）
- **发源地**：普宁市五峰山寒妈径
- **入海点**：汕头市潮阳区海门湾入南海
- **干流长度**：72 km
- **集雨面积**：1353 km²
- **上游别名**：上游普宁段称"流沙河"，汇合白坑湖水后正式称"练江"
- **经纬度范围**：约 116.0°–116.7°E, 23.2°–23.5°N
