# 茅洲河 — 上下游核对

## 水系地位

- **所属水系**：珠江三角洲独立入海小河系
- **地位**：深圳、东莞两市界河，深圳第一大河
- **发源地**：深圳羊台山北麓
- **入海点**：宝安区沙井街道民主社区注入珠江口伶仃洋
- **干流长度**：41.61 km（干流 31.3 km）
- **流域面积**：398.13 km²（深圳 310.85 + 东莞 77~87）
- **经纬度范围**：约 113.7°–114.0°E, 22.6°–22.85°N
