# 螺河（陆丰） — 上下游核对

## 水系地位

- **所属水系**：独流入海（粤东沿海诸河之一，陆丰市母亲河）
- **发源地**：陆河县与紫金县交界的三神凸山（海拔 1131 m）东坡
- **入海点**：陆丰烟港（碣石湾）注入南海
- **干流长度**：102 km
- **集雨面积**：1356 km²
- **年径流量**：23.6 亿 m³
- **经纬度范围**：约 115.3°–116.0°E, 22.8°–23.5°N
