# 珠江水系 / 西江 — 上下游核对

## 水系地位

- **所属水系**：珠江流域主干，珠江三大水系之一（西江、北江、东江）
- **发源地**：云南沾益马雄山（南盘江），经贵州、广西，至广东封开入省
- **入海/汇入点**：经肇庆、佛山三水思贤滘分流，由磨刀门、虎跳门等八大口门入海
- **干流长度**：2,214 km（全长）；广东段约 197 km（封开→三水思贤滘）
- **集雨面积**：353,120 km²（全流域）；广东境内约 22,430 km²
- **经纬度范围**（广东段 + 三角洲）：约 110.7°–113.5°E, 22.0°–23.5°N
- **同名辨析**："浔江"为西江在广西桂平—梧州段的称谓，广东境内不再使用
