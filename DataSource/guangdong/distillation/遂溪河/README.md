# 遂溪河 — 上下游核对

## 水系地位

- **所属水系**：粤西沿海诸河，**独流入海**
- **又名**：西溪河
- **发源地**：廉江市石城镇荔枝坑村东北独牛岭
- **入海点**：湛江市坡头区官渡镇石门村注入五里山港（雷州半岛东岸）
- **干流长度**：80 km
- **集雨面积**：1486 km²
- **年均径流量**：14.07 亿 m³
- **经纬度范围**：约 110.0°–110.5°E, 21.1°–21.7°N

## 数据重建记录（2026-04-26）

### 原始问题

v3 数据由 OSM bootstrap 生成，干流 13 段 + 26 条无名段。坐标核查：23 个文件全部在流域 bbox 内，无错归。

遂溪河为粤西小河，天地图 1:100万 和 OSM 覆盖均有限。干流数据来自 OSM 自动匹配，支流数据主要为无名段。
