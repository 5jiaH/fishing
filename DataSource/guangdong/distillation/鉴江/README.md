# 鉴江 — 上下游核对

## 水系地位

- **所属水系**：独流入海（广东省第三大独立水系、粤西沿海最大河流）
- **发源地**：信宜市云开大山虎豹坑
- **入海点**：吴川入南海
- **干流长度**：232 km
- **集雨面积**：9464 km²
- **流经区域**：信宜 → 高州 → 化州 → 吴川
- **经纬度范围**：约 110.5°–111.2°E, 21.2°–22.5°N
