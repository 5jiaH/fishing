# 韩江水系 — 上下游核对

## 水系地位

- **所属水系**：韩江水系（广东第二大水系）
- **发源地**：梅江（西源，发源于广东紫金白山栋）+ 汀江（东源，发源于福建宁化）
- **汇合点**：大埔县三河坝（梅江 + 汀江合流后始称韩江）
- **入海口**：潮州分流，经韩江东溪、西溪、梅河等入南海
- **干流长度**：约 470 km（含梅江上源）
- **集雨面积**：30,112 km²
- **经纬度范围**：约 115.0°–117.0°E, 23.3°–25.0°N
