# 黄冈河 — 上下游核对

## 水系地位

- **所属水系**：独流入海（粤东沿海诸河之一）
- **发源地**：饶平北部与福建诏安交界山区
- **入海点**：饶平县黄冈镇石龟头注入南海（柘林湾）
- **干流长度**：88 km
- **集雨面积**：1621 km²
- **年径流量**：16.4 亿 m³
- **流经区域**：上饶 → 饶洋 → 新丰 → 三饶 → 汤溪 → 浮山 → 浮滨 → 樟溪 → 高堂 → 联饶 → 黄冈 11 镇
- **经纬度范围**：约 116.6°–117.1°E, 23.5°–24.0°N
