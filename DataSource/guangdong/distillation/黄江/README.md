# 黄江（海丰） — 上下游核对

## 水系地位

- **所属水系**：粤东沿海诸河，**独流入海**
- **发源地**：莲花山脉腊烛山
- **入海点**：海丰县盐屿山注入南海（红海湾）
- **干流长度**：67 km
- **集雨面积**：1370 km²（省水利厅口径 1359 km²）
- **年径流量**：26 亿 m³
- **经纬度范围**：约 115.0°–115.7°E, 22.85°–23.2°N
