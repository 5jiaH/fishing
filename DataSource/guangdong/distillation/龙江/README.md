# 龙江（惠来） — 上下游核对

## 水系地位

粤东沿海诸河之一，**独流入海**。也称隆江河、龙溪。发源于普宁市大南山南水凹村附近，上游称**龙潭河**，流经陆丰市桂坑进入惠来，经普宁、陆丰、惠来三市县，于惠来县**神泉港**注入南海（1979 年改于南海哨所出海）。

- 集雨面积：1164 km²
- 干流长：82–88 km
- 年径流量：17.6 亿 m³
- 平均坡降：1.63‰

> **注意**：本文件夹 `龙江` 指**粤东潮汕的惠来龙江（独流入海）**，**不是**粤西阳江的"漠阳江 / 龙江段"，也不是"东江支流龙江"，也不是佛山顺德的"龙江大涌"。

### 关于 1:25 万数据

天地图 WFS (`gisserver.tianditu.gov.cn/TDTService/wfs`) 仅提供 1:100 万比例尺数据。1:25 万数据原由 webmap.cn（全国地理信息资源目录服务系统）提供分幅下载，但该系统已停止接受新注册，数据迁移至 `cloudcenter.tianditu.gov.cn`。

由于 1:25 万官方数据无公开 API 可用，本次使用 **OSM Overpass API** 补全缺失支流。OSM 数据的实际分辨率通常等效于 1:5 万~1:25 万，适合补充天地图 1:100 万缺失的小型支流。
